Timestamp API Microservice

This app is simple to use: simply request a date or unix timestamp, and the API will return a JSON object of both the natural language date and the unix time.

Examples of use:

https://band-point.glitch.me/Dec 31, 1999
https://band-point.glitch.me/946598400

Both of these queries will return the same JSON:

{"unix":946598400,"natural":"December 31, 1999"}
